# 第09章 文本分类与情感分析 —— 总结

> 把海量文本按需分类。**客观性内容分类** + **主观性观点(情感/情绪)分析**;方法从规则/统计到 BERT/LLM。

---

## 一、问题提出

### 文本信息分类两大类
1. **客观性事实/内容(facts/contents)** → **文本分类**(如《中国图书分类法》22 大类)。
2. **主观性观点(opinions)** → **观点挖掘(opinion mining)**:
   - **情感(sentiment)倾向性**:正面 positive / 负面 negative / 中立 neutral。
   - **情绪(emotion)**:7 类(喜怒忧思悲恐惊);数据集有 6/8/9/11/13 类不等(愤怒/厌恶/恐惧/喜悦/悲伤/惊讶/信任/期待/爱…)。

### 情感 vs 情绪
- **情感(sentiment)**:对事物是否满足需要的态度体验(喜好倾向)。
- **情绪(emotion)**:主观认知经验 + 行为反应(具体情绪类别)。

### 研究意义
自动分类提效;情报筛选推荐;企业观点挖掘/在线名誉管理;用户个性化服务;政府舆情监测与社会治理。

---

## 二、分类方法

### 总框架
**知识工程方法(基于规则)** + **机器学习方法**:
- I. 统计模式分类(特征表示 → 特征选择 → 分类器)
- II. 神经网络分类(Softmax 回归 / 预训练语言模型)

### 1. 特征表示
- **向量空间模型 VSM**(离散符号,TF-IDF 权重);**分布式表示**(词/句/文档向量)。

### 2. 特征选择 ⭐(4 种)
设特征 $t_i$ 与类别 $c_j$ 的 2×2 统计:$A_{ij}$(含特征且属该类)、$B_{ij}$(含特征不属该类)、$C_{ij}$(不含特征属该类)、$D_{ij}$(不含特征不属该类),$N_{\text{all}}$=总文档数,$C$=类别数。

| 方法 | 思想/公式 |
|------|----------|
| **文档频率 DF** | 特征在某类出现文档数;出现多则保留 |
| **互信息 MI** ⭐ | $I(X,Y)=H(X)-H(X\|Y)=\sum p(x,y)\log\dfrac{p(x,y)}{p(x)p(y)}$;$\text{MI}(t_i,c_j)=\log\dfrac{P(t_i,c_j)}{P(t_i)P(c_j)}$ |
| **信息增益 IG** ⭐ | $\text{IG}(C,T_i)=H(C)-H(C\|T_i)=-\sum p(c_j)\log p(c_j)+p(t_i)\sum p(c_j\|t_i)\log p(c_j\|t_i)+p(\bar t_i)\dots$ |
| **卡方 CHI** ⭐ | $\chi^2(t_i,c_j)=\dfrac{N_{\text{all}}(A_{ij}D_{ij}-C_{ij}B_{ij})^2}{(A_{ij}+C_{ij})(B_{ij}+D_{ij})(A_{ij}+B_{ij})(C_{ij}+D_{ij})}$,越大越相关 |

> 概率做**加 1 平滑**(如 $P(c_j\|t_i)\approx\dfrac{A_{ij}+1}{A_{ij}+B_{ij}+C}$)。例:"计算机" IG=0.1308 > "北京" IG=0.0293。

### 3. 分类器
- **生成式**:朴素贝叶斯 NB(最大似然/最大后验)。
- **判别式**:线性判别、**SVM**(最大间隔)、**最大熵 ME**、**CRF**。
- **组合分类器**:多特征 + 多分类器投票/加和/最大/乘法。

### 4. 神经网络分类
**① 基于 Softmax 回归**:
- 文档表示 $e_D$ → 全连接 $x=We_D+b$(维度=类别数 C)→ Softmax:
$$p(L(D)=c|x)=\frac{\exp(x_c)}{\sum_{c'}\exp(x_{c'})}$$
- 训练目标**最小化交叉熵**:
$$\text{Loss}=-\sum_{D\in T}\sum_{k=1}^{C}L_k(D)\log p_k(D)$$
($L_k(D)=1$ 若 D 属第 k 类,梯度下降训练 W、b)

**② 基于预训练语言模型(BERT)** ⭐:
- **预训练**(自监督):掩码语言模型 MLM + 句子衔接预测 NSP(见第7章)。
- **微调**:在 [CLS] + 文本 后加任务层(线性 + Softmax),用少量标注数据微调。句子级情感、篇章级内容分类均可。

## 三、情感与情绪分析

### 任务划分
1. **文本情感分析**:① 文档/句子情感识别 ② **要素级情感分析(ABSA)**(针对 aspect)③ 观点要素抽取(持有者/目标抽取、摘要)④ 观点检索。
2. **文本情绪分析**:情绪分类、情绪溯因、情绪-原因对抽取、情绪强度检测。

### 层次(词/特征/句子/文档级)
词级→特征/要素级(aspect,如"价格")→句子级→文档级(考虑篇章结构)。

### 情感识别方法
**词汇级**:
- ① **基于词典**:用 WordNet/HowNet 的同义反义关系扩展情感词典。
- ② **基于语料**:$SO(\text{word})=\text{PMI}(\text{word,"excellent"})-\text{PMI}(\text{word,"poor"})$。

**句子级**:
- A. 基于词典(词性标注选情感短语 → PMI 算极性 → 平均)。
- B. 基于统计学习(uni/bi-gram、POS、形容词特征;NB/SVM/ME;组合分类器)。
- C. 基于 Softmax / 预训练语言模型(BERT)。

> **组合学习**:多特征(POS/依赖)× 多分类器(NB/SVM/MaxEnt)→ 集成打分,优势互补。

## 四、基于 LLMs 的情绪支持对话(ESC)

- **情绪支持代理(Emotional Support Agent)**:缓解用户情绪困扰,用于社交、客服、心理健康。
- **流程 + 策略(8 种支持策略)**。
- **LLM 局限**:缺乏共情、回复过长、程式化、数据多样性不足、策略预测不准。
- **改进方法**:用先进 LLM 增强 ES 数据——**Completion / Rewriting / Simulation**;角色扮演对话生成(Seeker/Counselor/Supporter 多视角),如 **SweetieChat**、**ServeForEmo**(45 类问题)。

## 五、资源与评测

### 情感词典
- **英文**:General Inquirer、SentiWordNet(基于 WordNet)、OpinionFinder、IBM Lexicon。
- **中文**:**HowNet**(6 子集:正/负情感、正/负评价、程度、主张)、**大连理工情感词汇本体库**。

### 评测会议
- **TREC Blog Track**(2006起,英文网页,观点检索+极性)。
- **NTCIR**(多语新闻,主题/观点识别/极性/持有者/目标抽取)。
- **COAE**(中文,词级/文档级 sub-obj、正负、目标抽取、检索)。

### 数据集
- 句子级:IMDb、**SST**(SST-2/SST-5)、Yelp、中文微博。
- 属性级 ABSA:**SemEval-14**(餐馆/笔记本,属性抽取/情感/类别)、Sentihood、ASAP(中文电商,18 属性)。

### 评价指标
分类:**准确率(Accuracy)、精确率 P、召回率 R、F1**;排序任务等。

### 二、公式 / 计算
- ⭐⭐ **互信息** $I(X,Y)=\sum p(x,y)\log\frac{p(x,y)}{p(x)p(y)}$。
- ⭐⭐ **信息增益** $\text{IG}=H(C)-H(C|T_i)$(展开三部分)。
- ⭐ **卡方** $\chi^2=\dfrac{N_{\text{all}}(AD-CB)^2}{(A+C)(B+D)(A+B)(C+D)}$。
- ⭐ Softmax 分类 $p=\exp(x_c)/\sum\exp$;交叉熵损失 $\text{Loss}=-\sum\sum L_k\log p_k$。
- 情感倾向 $SO(\text{word})=\text{PMI}(\text{word,excellent})-\text{PMI}(\text{word,poor})$。
- 2×2 列联表(A/B/C/D)的概率估计(含 +1 平滑)。
- 会手算 IG(如"计算机"特征)。

## 本章小结(背诵要点)

- **分类**:客观内容分类 + 主观情感/情绪分析;情感(倾向)vs 情绪(类别)。
- **分类流程**:特征表示(VSM/分布式)→ **特征选择(DF/MI/IG/CHI)** → 分类器(生成式 NB / 判别式 SVM·ME·CRF)→ 组合分类。
- **神经网络分类**:**Softmax 回归**(交叉熵损失)+ **预训练模型微调**(BERT:MLM+NSP 预训练,任务层微调)。
- **情感分析**:任务=情感识别 / **ABSA 要素级** / 观点要素(持有者+目标)抽取 / 观点检索;层次=词/句/文档/aspect;方法=词典(WordNet/HowNet)/语料($SO=\text{PMI}_{pos}-\text{PMI}_{neg}$)/统计(NB·SVM)/神经网络;**组合学习**优势互补。
- **ESC**:LLM 情绪支持对话,角色扮演数据增强(SweetieChat/ServeForEmo)。
- **资源**:词典(HowNet、SentiWordNet)、评测(TREC/NTCIR/COAE)、数据集(SST/IMDb/SemEval-14 ABSA)。
