## 1. 依存图是什么

依存语法不把句子拆成一层层短语（NP/VP/PP），而是**直接在"词"之间画箭头**：

- 每个箭头 $A\to B$ 表示 **$A$ 支配 $B$、$B$ 从属于 $A$**（$A$ 是 head 支配词，$B$ 是 dependent 从属词）。
- 箭头上标**关系类型**（主谓、动宾、介宾……）。
- **整句只有一个根**，根是句子的**核心动词**——动词支配一切、自己不被支配。
- 所有词（除根外）都有且只有一个"父亲"。

> **和短语结构树对比**：短语结构树讲"包含"（NP 里有什么），依存树讲"谁管谁"（哪个词是中心）。依存树是**平的、词汇化**的——每个节点就是一个真实词。
>
> **背景**：法国语言学家 **Tesnière（泰尼埃）**创立，《结构句法基础》(1959/1965)；核心三概念：关联、组合、转位；**动词是句子中心**。**价(valency)**：动词能支配的行动元个数 → 零价(地震)、一价(休息)、二价(爱)、三价(给)。

## 2. 画图的两条铁律：找中心词 + Robinson 四公理

### 规则 A：找中心词（head）—— 转换的关键

每个短语都有一个**中心词**，短语在依存树里就用它的中心词"代表"。确定方法：

| 短语类型 | 中心词 |
|---|---|
| **NP**（名词短语）| 名词 |
| **VP**（动词短语）| 动词 |
| **PP**（介词短语）| 介词 |
| **S**（整句）| 主要动词 |

**转换口诀**：自底向上，每个短语的中心词去支配它子成分的中心词；最终整个 $S$ 的中心词（主动词）就是 ROOT。

### 规则 B：Robinson 四公理（1970）—— 自检用 ⭐

画完一定要用这四条检查，等价于 **"单一父结点 + 连通 + 无环 + 可投射 = 一棵有根树"**：

1. 一个句子**只有一个**独立成分（ROOT）；
2. 其他成分都从属于某个成分；
3. 任何成分**不能依存于两个或多个**成分（单一父结点）；
4. **可投射性（projective）**：$A$ 从属于 $B$，中间夹着的 $C$ 必须也从属于 $A$、$B$ 或二者之间——直观说就是**依存弧不能交叉**。

> **投射 vs 非投射**：依存弧是否与其他词相交。规范汉语/英语多为投射；决策式移进-归约算法**不能处理非投射**，这是其局限之一。

## 3. 完整例子：把短语结构树转成依存图

### 句子：他在阳台读书。

短语结构树（题目给定）：

```
                    S
         ┌──────────┴──────────┐
        NP                    VP
        他            ┌────────┴────────┐
                    PP                 VP
              ┌──────┴──────┐     ┌────┴────┐
              P            NP     V        NN
              在          阳台    读        书
```

### 第 1 步：标出每个短语的中心词

- $S\to$ 中心是主动词 **读**（$=$ ROOT）
- $NP(\text{他})\to$ **他**
- $VP(\text{在阳台\ 读书})\to$ 中心是其右子 $VP$ 的中心 $=$ **读**
- $PP(\text{在阳台})\to$ 中心是介词 **在**
- $NP(\text{阳台})\to$ **阳台**
- $VP(\text{读书})\to$ **读**
- $V(\text{读})\to$ **读**；$NN(\text{书})\to$ **书**

### 第 2 步：自底向上画弧

- **主语**：$NP(\text{他})$ 是 $S$ 的主语 → **他 ← 读**（读支配他）
- **状语**：$PP(\text{在阳台})$ 作处所状语修饰动词读 → PP 的中心词 **在** 从属于 **读**
- **介词内部**：**在** 支配其宾语 **阳台**
- **宾语**：$VP(\text{读书})$ 里 **书** 从属于动词 **读**

### 第 3 步：标关系类型，成树

```
                    读  (ROOT)
                 /   /    \    \
               他   在     书    。（标点）
                    |
                   阳台
```

| 依存弧（支配 → 从属）| 关系类型 | 含义 |
|---|---|---|
| 读 → 他 | **SBV**（主谓 / nsubj）| 他是读的人（施事）|
| 读 → 在 | **ADV**（状中 / advmod）| "在阳台"作处所状语修饰读 |
| 在 → 阳台 | **POB**（介宾 / obl）| 阳台是介词在的宾语 |
| 读 → 书 | **VOB**（动宾 / obj）| 书是被读的对象（受事）|

### 第 4 步：用四公理自检 ✓

- 只有一个根 `读` ✓；所有词都有父（他/在/书 → 读，阳台 → 在）✓；每个词单一父结点 ✓；弧不交叉（可投射）✓。

> **关键易错点**：很多人会把"阳台"直接连到"读"上，跳过了介词"在"。**错！** 介宾短语里，介词"在"才是短语的中心，"阳台"先从属于"在"，"在"再整体从属于"读"——这是 PP 转换的核心。记住 **PP 的 head 是介词，不是它后面的名词**。

## 4. 常见汉语依存关系标签（背这几组够用）

| 关系 | LTP 标签 | UD / 英文 | 例子 |
|---|---|---|---|
| 主谓 | **SBV** | nsubj | **他** → 读 |
| 动宾 | **VOB** | obj | 读 → **书** |
| 介宾 | **POB** | obl / case | 在 → **阳台** |
| 状中（状语修饰中心）| **ADV** | advmod | **认真** → 读 / 在阳台 → 读 |
| 定中（定语修饰中心）| **ATT** | amod / nmod | **红色的** → 苹果 |
| 动补 | **CMP** | xcomp / compound:vc | 洗 → **干净** |
| 并列 | **COO** | conj | 苹果 **和** 橘子 |
| 核心（根）| **HED** | root | （虚拟根 → 读）|

> **记忆诀窍**：**"中"那个词是被修饰的中心**。状**中** = 状语修饰中心动词；定**中** = 定语修饰中心名词；关系箭头**从中心词指向修饰语**（中心是 head，修饰语是 dependent）。

## 5. 巩固练习（两个小例子）

**例 1「他认真读书。」**（状语是简单副词）

```
              读 (ROOT)
           /   /    \
         他  认真    书
```

读 → 他(SBV)、读 → 认真(ADV)、读 → 书(VOB)。

**例 2「红色的苹果很甜。」**（含定语 + 形容词谓语）

```
              甜 (ROOT)
             /    \
          苹果    很
           |
         红色的
```

- 根是形容词谓语 **甜**（没有动词时，谓语形容词/名词也能当根）；
- 苹果 → 甜(SBV)；很 → 甜(ADV)；**红色的 → 苹果(ATT)**——定语挂在中心名词上，不是挂在谓语上。

## 6. B 卷第 3 题完整作答（分析标注题，10 分）

> **题目**：给出短语结构树（"他在阳台读书。"），(1) 画出依存关系图；(2) 标出该句子的语义角色。

### (1) 依存关系图

以核心动词 **`读`** 为根（ROOT），依存弧（支配词 → 从属词，括号内为关系类型）：

```
                    读  (ROOT)
                 /   /    \    \
               他   在     书    。
                    |
                   阳台
```

- `读 → 他`　主谓关系（SBV / nsubj）
- `读 → 在`　状中关系（ADV / advmod）—— 介宾短语"在阳台"作处所状语修饰"读"
- `在 → 阳台`　介宾关系（POB / obj）—— "阳台"依存于介词"在"
- `读 → 书`　动宾关系（VOB / obj）

满足 Robinson 四公理：单一父结点、连通、无环、可投射，是一棵有根树。

### (2) 语义角色标注

以谓词 **`读`** 为核心：

| 成分 | 语义角色 | 说明 |
|------|----------|------|
| `他` | **ARG0**（施事 Agent）| 读书的人 |
| `书` | **ARG1**（受事 / 客体 Patient）| 被读的对象 |
| `阳台`（在阳台）| **ARGM-LOC**（地点）| 动作发生处所 |

> **联动技巧**：依存图的 ROOT/ARG0/ARG1 往往就是语义角色题的谓词和施事/受事——先画好依存图，语义角色基本就能顺出来。

## 🎯 考试提示

- **题目套路**：给短语结构树 → ① 画依存图 ②（常配套）标语义角色。两题联动。
- **作答三步**：找主动词当 ROOT → 逐短语找中心词画弧 → 标关系类型。
- **必查**：① 只有一个根；② 介词短语别把名词直接挂动词上（**PP 的 head 是介词**）；③ 弧不交叉（可投射）。
- **等价标签都给分**（SBV/nsubj、ATT/nmod 都行），但**关系判对比标签写法重要**——主谓/动宾/状中/介宾别标反。
- **评价指标**（简答可能考）：**UAS** = 找到正确支配词的词占比（含根）；**LAS** = 支配词**和**关系类型都对（含根）；**DA** = 非根词中正确占比。英文规范文本 UAS 90%+。
